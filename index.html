Modern Web Dev Workflow

💻

Local Git Repository

Write code in VS Code → track changes with git init, git add ., git commit. Every save is versioned locally.

↓
🐙

GitHub (Remote Repository)

Push code with git push origin main. GitHub stores the cloud backup and triggers CI/CD pipelines on every push.

↓
⚡

CI/CD — GitHub Actions

A .yml workflow runs automatically: linting (ESLint) + HTML validation + tests to catch bugs before they go live.

↓
🌍

Hosting (Website + Database)

Frontend → GitHub Pages (free). Backend API → Render (free tier). Database → PostgreSQL on Render. All automated!

↓
🌐

Web Browser

The live portfolio is accessible to everyone on the internet. Contact form submits → Express API → stored in PostgreSQL. 🎉
